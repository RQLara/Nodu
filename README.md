Nodu
Web de asesoría gastronómica
Descripción
Plataforma web desarrollada para un asesor gastronómico independiente. Permite a cualquier usuario reservar una cita, enviar una idea de negocio y consultar proyectos y artículos del blog. El asesor gestiona todo desde un panel privado de administración.
Tecnologías

HTML5, CSS3 y JavaScript
Supabase (base de datos, autenticación y almacenamiento)
Git y GitHub

Funcionalidades

Reserva de citas
Cajón de ideas
Proyectos
Blog
Panel de administración privado
